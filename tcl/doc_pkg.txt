3.ntp installation

Install Chrony
sudo apt install chrony -y
3. Enable and Start Chrony
sudo systemctl enable chrony
sudo systemctl start chrony
4. Check Chrony Status
sudo systemctl status chrony

You should see:

active (running)

Press:

q

to exit status screen.

5. Verify Time Synchronization
chronyc tracking

and

chronyc sources -v
6. Check Current Time
timedatectl

Expected:

System clock synchronized: yes
NTP service: active
7. Configure Timezone (Optional)

List timezones:

timedatectl list-timezones

Set India timezone:

sudo timedatectl set-timezone Asia/Kolkata

Verify:

timedatectl
8. Open Firewall for NTP (If Server)

If Ubuntu firewall is enabled:

sudo ufw allow 123/udp

Reload:

sudo ufw reload
